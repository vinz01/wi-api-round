# wi-api-round

Password manager web app:

Tech stack: Django, MySQL

Endpoints: 

Home: app


Register: app/user


Login: app/user/auth


Panel: app/panel


Add Note: app/sites/<int: userid>
  

View Notes: app/sites/list/<int :userid>
